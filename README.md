# ⏱️ Zeiterfassungs-App
Eine öffentliche App zur Zeiterfassung in ERPNext/Frappe HRMS.
Die App dient der präzisen und benutzerfreundlichen Erfassung von Arbeitszeiten auf Projektebene und Aufgabenbasis. Sie richtet sich an Teams und Unternehmen, die eine integrierte Lösung für Zeiterfassung und Arbeitszeitkontrolle benötigen.
